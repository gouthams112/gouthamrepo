METADATA tags:

  1. < Meta >: meta tag is an HTML tag that provides metadata about a web page.

  2. < charset&g:The charset tag is an HTML meta tag that specifies the character encoding used by a web page.

  3. < X-UA-Compatible>: Is an HTML meta tag that is used to specify the version of Internet Explorer (IE) that a web page is optimized for.

  4. < IE=edge>:is used the latest edge versions of rendering engine.

  5. < Viewport>:The viewport refers to the visible area of a web page within a web browser window.

  6. < width=device-width>:is an attribute of the 'viewport' meta tag that is used to set the width of the viewport to the width of the device's screen.

  7. < initial-scale=1.0 >: Meta tag that is used to set the initial zoom level of a web page on a mobile device.
