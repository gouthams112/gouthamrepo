CSS Syntax

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in HTML or XML. The basic syntax for CSS is as follows:

  • Selector: In CSS, selectors are used to select the HTML elements that you want to style. A selector can be a tag name, a class, an ID, or a combination of these.
  • Property:In CSS, properties are used to define the visual appearance of an HTML element. They specify the characteristics of an element, such as its color, size, font, margin, padding, and more.
  • Value:In CSS, values are used to define the specific characteristics of an HTML element that are set by a property. A value is the actual setting that is applied to a particular property.

    • The basic of selector

    1. Element Selector:

      2. An element selector is used to select and apply styles to HTML elements on a web page.It specify the name of HTML element.

    2. Group Selector:

      3. A group selector is a way to select multiple elements at once and apply the same styles to all of them. The syntax for a group selector is to list the selectors for each element separated by a comma.

    3. Universal Selector:

      4. The universal selector is a selector that matches all elements on a web page. The universal selector is represented by an asterisk (*).
