Box Model

When we are talking about the layout and design we see box model.The CSS Box Model is a box that wrapsaround every HTML Elements.In box model it consists of

  1. Content
  2. Padding
  3. Border
  4. Margin

1.Content

It refers to the actual content of an HTML element, such as text, images, or other types of media that are contained within the element.

2.Padding

It refers to the space between the content of an HTML element and its border. It clears the area between the content.the padding is transparent.

3.Border

It refers to the line or outer edge that surrounds the padding and content of an HTML element.

4.Margin

It clears the area outside the border. The margin is transparent.
