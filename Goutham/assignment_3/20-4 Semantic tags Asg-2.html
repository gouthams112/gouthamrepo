Semantic Tags

Semantic tags are HTML tags that provide meaning and context to the content within them, rather than just specifying how the content should look.They help search engines, screen readers, and other tools to better understand the content and its structure.

  • < p > : tag in HTML is used to define a paragraph.

  • < h1 > to < h6 > : tags in HTML are used to define headings of different levels.

  • < a > :tag in HTML is used to create a hyperlink to another webpage, document, or a specific location within a webpage.

  • < ul > : tag in HTML is used to create an unordered list of items.

  • < ol > : tag in HTML is used to create an ordered list of items.

  • < li > : tag in HTML is used to define a list item within an ordered or unordered list.

  • < img > :tag in HTML is used to embed an image in a webpage.
