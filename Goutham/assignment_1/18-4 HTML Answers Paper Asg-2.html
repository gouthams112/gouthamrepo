  1. What html stands for?
    Ans.HTML stands for HyperTextMarkupLanguage.
  2. Define is html?
    Ans.HTML(HyperTextMarkupLanguage)is a markup language used to create and structure content for the World Wide Web. It consists of a set of markup tags and attributes that are used to define the structure and layout of a web page.HTML code is interpreted by web browsers to render web pages with text, images, videos, and other multimedia elements. HTML provides the foundation for the visual appearence and functionality of web pages, and is often combined with other web technologies such as CSS and JavaScript to create interactive and dynamic web content
  3. what are the tags in HTML?
    AnsTags in HTML are used to define elements and their properties on a web page. They are enclosed in angle brackets (<>) and usually come in pairs, with the opening tag indicating the start of an element and the closing tag indicating the end. Some common HTML tags include:
      <html>: Defines the beginning and end of an HTML document
      <head>: Contains meta information about the document, such as the title and links to stylesheets
      <body>: Contains the visible content of the document
      <h1> to <h6>: Heading tags used to define headings of different levels
      <p>: Defines a paragraph of text
      <ul> and <li>: Used to create unordered lists
      <ol>and <li>: Used to create ordered lists
  4. What is the purpose of section in an HTML document?
    Ans'<section>'tag in HTML is used to group together related content on a web page. It is a semantic element that helps to structure the page and indicate the purpose of the content within it.
  5. What are ordered list and unordered list in HTML?
    Ans HTML, ordered and unordered lists are used to group together related items or pieces of information. The main difference between the two types of lists is the way in which the items are numbered or bulleted. An unordered list, created using the <ul> tag, consists of a series of items that are each preceded by a bullet point. On the other hand, an ordered list, created using the <ol> tag, consists of a series of items that are each preceded by a number or letter, depending on the type of list.
